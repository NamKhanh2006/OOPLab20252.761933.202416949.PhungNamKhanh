Questions:
(Part 6.1)
- What happens if users choose “Cancel”?
- How to customize the options to users, e.g. only two options: “Yes” and “No”, OR “I do” 
and “I don’t” (Suggestion: Use Javadocs or using Eclipse/Netbean IDE help).  

Answer:
- In the standard JOptionPane.showConfirmDialog implementation used in this exercise, choosing "Cancel" returns a specific integer value (typically 2 or JOptionPane.CANCEL_OPTION).If the program only checks for a "Yes" (usually 0) and assumes everything else is a "No," clicking "Cancel" will often trigger the same logic as clicking "No". When the source code provided is compiled and run, a "Cancel" selection from the user displays the message: "You've chosen: No", which is the same message that is displayed when the user chooses "No".
- To customize button text, we must use a more advanced version of the showOptionDialog method found in the javax.swing.JOptionPane class. For example:
+ To change to just "Yes" and "No": Instead of the default YES_NO_CANCEL_OPTION, we can specify the YES_NO_OPTION constant. This is done by changing the line:
int option = JOptionPane.showConfirmDialog(null,
				"Do you want to change to the first class ticket?");
in the code to this:
int option = JOptionPane.showConfirmDialog(null, 
    "Do you want to continue?", 
    "Selection", 
    JOptionPane.YES_NO_OPTION);
+ To use custom text like “I do” and “I don’t”: We must pass an array of Object (usually Strings) as the options parameter to showOptionDialog. This is done by changing the line:
int option = JOptionPane.showConfirmDialog(null,
				"Do you want to change to the first class ticket?");
in the code to this:
Object[] options = {"I do", "I don't"};
int n = JOptionPane.showOptionDialog(null,
    "Do you want to continue?",
    "A Custom Question",
    JOptionPane.YES_NO_OPTION,
    JOptionPane.QUESTION_MESSAGE,
    null,     // No custom icon
    options,  // The titles of buttons
    options[0]); // Default button title
- Tip: Using Javadocs or IDE Help to find this
+ In Eclipse/NetBeans: We can hover our mouse over showConfirmDialog or JOptionPane and press F2 (or Ctrl+Click) to see the built-in documentation and see other available methods.
+ In Javadocs: We can navigate to the javax.swing package and click on the JOptionPane class to view all available "Method Summaries" and constant definitions.
